```
int hci_trans_h4_recv_byte(uint8_t byte);
```



H4 Type: 0x1 - 0x5

Success:

```
command
-> 0x01 0x3c 0x0f 0x00
<- HM_SUCCESS
-> 0x01 0x3c 0x0f 0x02 0xfc 0xfc
<- HM_SUCCESS

acl
-> 0x02 0x01 0x00 0x00 0x00 
<- HM_SUCCESS
-> 0x02 0x01 0x00 0x01 0x00 0x06 0x06
<- HM_SUCCESS & HM_H4_SYNC_ERROR
-> 0x02 0x01 0x00 0x00 0x01 (0x06 repeat 256)
<- HM_SUCCESS

sco
-> 0x03 0x01 0x00 0x00
<- HM_SUCCESS
-> 0x03 0x01 0x00 0x02 0xfc 0xfc
<- HM_SUCCESS

event
-> 0x04 0x80 0x00
<- HM_SUCCESS
-> 0x04 0x80 0x02 0xfc 0xfc
<- HM_SUCCESS

iso
-> 0x05 0x01 0x00 0x00 0x00
<- HM_SUCCESS
-> 0x05 0x01 0x00 0x02 0x00 0xfc 0xfc
<- HM_SUCCESS
-> 0x05 0x01 0x00 0x02 0xc0 0xfc 0xfc
<- HM_SUCCESS
```



Fail:

```
-> 0x06
<- HM_H4_SYNC_ERROR

-> 0x50
<- HM_H4_SYNC_ERROR

-> 0xFF
<- HM_H4_SYNC_ERROR
```
